My First Blog Post

Published on May 13, 2025 by Okech Odhiambo

This marks the beginning of my journey into blogging and sharing my experiences with web development. Creating this very website was a fantastic learning opportunity, reinforcing the core concepts of HTML, CSS, and JavaScript.

The Building Blocks: HTML

Semantic HTML was key. Using tags like <header>, <nav>, <main>, <article>, and <footer> not only helps structure the content logically but also improves accessibility and SEO.

Making it Pretty: CSS

CSS brought the structure to life. Flexbox was used for layout, particularly in the header for aligning the logo and navigation. Media queries ensure the site adapts to different screen sizes, making it responsive. Basic styling like colors, fonts, padding, and margins create a pleasant visual experience.


/* Basic responsive navigation toggle */
@media (max-width: 768px) {
  .main-nav {
    /* Styles to hide/show on mobile */
  }
  .nav-toggle {
    display: block; /* Show hamburger */
  }
}

Adding Interactivity: JavaScript

JavaScript adds the dynamic elements. A simple script handles the mobile navigation toggle (the "hamburger" menu) and provides basic client-side validation for the contact form, preventing submission if fields are empty.

Deployment

Finally, deploying the site using a static hosting service like GitHub Pages, Netlify, or Vercel makes it accessible to the world. This process is surprisingly straightforward for static sites.

It's been a rewarding process, and I look forward to adding more content and features to this blog!
